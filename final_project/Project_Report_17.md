# A second Agent

I did some research to find resources on logical fallacies and found a site with over 150 of them. The primary chat agent uses the second Agent, called Devil's Advocate to assist in identifying logical fallacies, weak arguments, logical holes or any other issues. This second agent packs a punch and helps the main agent find errors in ways that it probably wouldn't have been able to otherwise. But I need to test to be sure.

# Prompt Engineering Adjustment

I also clarified the prompts that the main agent which improves the User experience within the app. It has a better flow now to help the user along.

# Playtesting

I had a friend of mine playtest it and they liked it. He said that it helped them think through their issue, even though they used a contrived example.
